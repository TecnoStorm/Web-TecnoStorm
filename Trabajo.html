¿Qué puestos están vacantes?

  • Gestor base de datos

    Si tiene experiencia con MySQL y MariaDB este es tu puesto de trabajo, ya que en el mismo se encargará como su nombre dice de gestionar las bases de datos de la empresa, con esto también nos referimos a saber realizar con éxito D.E.R y como normalizarlos

    Si tienes conocimientos sobre CSS y html este puesto será de su agrado ya que en el mismo se encagarán de diseñar y realizar las páginas web que realize la empresa

    Los proyectos de la empresa ambarcan también scripts de todo tipo, como por ejemplo un script gestor de usuarios un script de puntuación, etc. Los requerimientos minimos para este empleo es conocer los comandos de la terminal de Linux ya sea para cambiar permisos, filtros y expresiones regulares

    La empresa a la hora de programa utilizará el lenguaje de php por ende si conoce y sabe utilizar este lenguaje este sera so puesto ideal en el cual trabajara con formularios, con POST,GET y todo lo que pueda ofrecer php
